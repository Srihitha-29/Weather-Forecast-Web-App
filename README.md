# 🌦️ Weather Forecast Web App

A modern and responsive weather forecasting web application built using **HTML, CSS, and JavaScript**.
The application fetches real-time weather information and a 5-day forecast using the **OpenWeatherMap API**, 
featuring a dynamic glassmorphism interface with temperature-based theme changes.

## ✨ Features

* 🔍 Search weather information for any city worldwide
* 🌡️ Real-time weather updates
* 💧 Humidity information
* 💨 Wind speed details
* 🌤️ Weather condition icons
* 📅 5-Day weather forecast
* 🎨 Dynamic temperature-based UI theme
* 💎 Modern glassmorphism design
* 📱 Fully responsive layout
* ⌨️ Enter key search support
* ⚠️ Error handling for invalid city names and network issues

### Temperature-Based Theme Colors

| Temperature Range | Theme              |
| ----------------- | ------------------ |
| ≤ 10°C            | 🔵 Cool Blue       |
| 11°C – 25°C       | 🟢 Fresh Green     |
| > 25°C            | 🔴 Warm Red/Orange |

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/Srihitha-29/Weather-Forecast-Web-App.git
```

### Navigate to Project Folder

```bash
cd Weather-Forecast-Web-App
```

### Run the Application

1. Open the project in **VS Code**
2. Install the **Live Server** extension (if not already installed)
3. Right-click on `index.html`
4. Select **Open with Live Server**
5. Enter any city name and view the weather forecast

## 🔑 API Information

This project uses the **OpenWeatherMap API**.

### Current Weather Endpoint

```text
https://api.openweathermap.org/data/2.5/weather
```

### Forecast Endpoint

```text
https://api.openweathermap.org/data/2.5/forecast
```

Get your free API key here:

👉 **[OpenWeatherMap API](https://openweathermap.org/api)**
